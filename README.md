
🎮 Trapstorm - A React Party Game
A fun, fast-paced party game built with React where players compete across three exciting mini-games! Perfect for groups, game nights, and testing your knowledge, creativity, and communication skills.
🌟 Features
Two Game Modes
Normal Mode: Play one game at a time at your own pace
Storm Mode: Chain all 3 games back-to-back for the ultimate challenge! ⚡
Three Mini-Games
1. 🎯 Name Drop
Test your knowledge and speed! Answer category-based questions by naming things that fit the criteria.
Example: [Gen Z] Name 5 ways you cope with decision overload
Gameplay: Timer-based rounds where players take turns naming items
Scoring: Points awarded for valid answers
2. 🔥 Roast Rumble
Debate abstract topics and justify your wild takes!
Example: "Justify why popcorn should be considered a meal"
Gameplay: Players debate and argue for/against random topics
Scoring: Vote-based scoring where others rate your argument
3. 🧠 Mind Meld
Describe words in 60 seconds WITHOUT saying them directly!
Example: Category: Cartoon Characters → Describe "Pikachu"
Gameplay: Give clues and descriptions while teammates guess
Scoring: Points for correct guesses within the time limit
